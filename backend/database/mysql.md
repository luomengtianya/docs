## sql使用

* 获取数据库中的表信息

```sql
SELECT TABLE_NAME AS NAME, TABLE_COLLATION AS COLLATION, TABLE_COMMENT AS COMMENT FROM `TABLES` WHERE TABLE_SCHEMA = 'user_manage_db' ORDER BY TABLE_NAME
```

* 获取数据库中表的字段信息

```sql
SELECT COLUMN_NAME AS NAME, COLUMN_DEFAULT AS `DEFAULT`, COLUMN_KEY AS `KEY`, IS_NULLABLE AS NULLABLE, COLUMN_TYPE AS TYPE, COLUMN_COMMENT AS `COMMENT`, EXTRA
FROM COLUMNS WHERE TABLE_NAME = 'sys_action' AND TABLE_SCHEMA = 'user_manage_db';
```
